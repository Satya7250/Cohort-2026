# Resume Website

## About This Project
This is a simple **single-page resume website** built using **only HTML**.
I made this project while learning web development in the **Chai aur Code cohort**.

The main goal of this project is to practice HTML basics and understand how to
structure a resume using web technologies.

---

## Tech Used
- HTML

This project does not use CSS or JavaScript.

---

## What This Page Contains
- Name and profile image
- About me section
- Skills list
- Education details
- Experience section
- Projects section
- Contact information
