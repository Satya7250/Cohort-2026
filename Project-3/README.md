# Cursor Landing Page

## Description
This project is a landing page clone of **Cursor**, an AI-powered coding editor.  
It presents the main product highlights, testimonials, features, and download call-to-action using a modern dark-theme design.

## Live Demo:  
https://cursor-sp.vercel.app/

---

## Project Structure

```
project-2/
├── index.html           
├── style.css            
├── README.md        
└── images/             
    ├── cursor_logo.png
    ├── cursor_tab_logo.png
    ├── product.png
    └── [Other brand logos]
```

| File | Description |
|------|-------------|
| `index.html` | Main HTML structure of the landing page |
| `style.css` | Styling, layout, and theme colors |
| `images/` | All images and logo assets |

---

## Features
- Navigation bar with dropdown menu
- Hero section with product preview
- Feature showcase sections
- Testimonials cards
- Footer with multi-column links
- Dark theme UI using CSS variables

---

## Technologies Used
- **HTML5** – Page structure  
- **CSS3** – Layout, styling, and theme variables  

---

## Responsiveness
This project is **partially responsive** and works best on desktop screens.  
Some sections may require additional media queries for full mobile responsiveness.

---

## Getting Started
To run locally:

```bash
git clone <repository-link>
cd project-folder
open index.html
```
